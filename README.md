# Family Budget Manager

A full-stack application for families to manage and track their shared budget items.

## Project Structure

- **Backend**: Node.js + Express API server
- **Frontend**: React application

## Getting Started

### Backend Setup

```bash
cd backend
npm install
node server.js
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

## Features

- User authentication
- Family management
- Budget item tracking
- Shared budget dashboard

## Tech Stack

- **Backend**: Node.js, Express
- **Frontend**: React
- **Database**: MongoDB (or your database)
